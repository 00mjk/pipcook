Interface DataProcessType

After the DataAccessType plugin unifies the data format and outputs the current training data, the DataProcessType Performs unified pre-processing operations before the data enters the model, including data cleaning, data transformation, and data standardization.

Theoretically, you can use this kind of plugin to process your data in any form, including changes to data features.

example
const doubleSize: DataProcessType = async (sample: Sample, metadata: Metadata, args?: ArgsType): Promise<void> => {
  // double the data
  sample.data = sample.data * 2;
};
export default doubleSize;
