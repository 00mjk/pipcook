Interface DataCollectType

The data collect plugin is designed to help users collect various data and store the data in a standardized way for subsequent plugins in the pipeline.

The sources of data can be various, such as files and folders in various local formats, files downloaded from the internet, and data queried from database. At the same time:

  • it should also support dividing data into different datasets and clearly revealing the data format.
  • it should accurately output information about the data itself, such as the name of each feature, the type of the feature, the number of samples, and all relevant meta information of the data.
example
const collectTextline: DataCollectType = async (args: ArgsType): Promise<void> => {
  const { uri, dataDir } = args;
  await fs.copy(uri, dataDir + '/my-own-dataset.csv');
  return null;
};
export default collectTextline;
